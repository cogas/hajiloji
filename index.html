ようこそ！このサイトは、ロジバン入門講座です！
ロジバンは人工言語の中でも、専門の文法用語の多さや、独特な文法構造のせいで難しいと思われがちです。
しかし、ロジバンは従来の自然言語の「贅肉」を取り払った、非常にシンプルな言語だと言えます！
このサイトでは、こしょん(kocon),そら（soran),セヴァン(sevan)といった女の子たちの会話をききながら、
楽しくロジバン文法の基礎を学ぶことができます。

  • The Complete Lojban Language 日本語抄訳 : 日本のロジバニスト精鋭が訳した、ロジバン文法の公式文書。

  • wikibooks ロジバン : おなじみのwikibooksです。適度にまとまっています。
  • wavelesson : 最新の文法が学べるテキストのひとつ。ただし英語。
  • lojban wavelesson 日本語訳 : wavelesson の日本語訳。
  • ロジバン語会話集 : おなじみのwikitravelです。travelだけあって、日常的な文が参考になります。
  • jboselkei : みんなでロジ作問題を出題しあって、ロジ文を書いていくというサイト。いい演習に。
  • 公式wiki 学習ページ : ロジバン公式wikiの学習ページです。さまざまな学習ツールへのリンクが張ってあります。

辞書とか単語集とか

  • jbovlaste : 読んで字のごとく「ロジバン辞書」。公式辞書です。
  • vlasisku : jbovlaste をより使いやすく、スタイリッシュにしたもの。英語。
  • sutsis : vlasiskuの軽量版です。
  • gismu liste : 日本語のgismu定義集。便利！「ctrl + f」でページ内検索かけましょう。
